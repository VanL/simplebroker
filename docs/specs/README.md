See `00-specs-index.md` for the canonical numbered entry point and reading

## Scope (this repository)

This spec tree currently governs the **development process** (the
operating model, [DOM-*]) — not product behavior. **SimpleBroker's
product behavior is specified by the root `README.md`**, per the
declared doctrine in `docs/README.md`; the CHANGELOG records behavior
deltas. Promoting README content into coded, spec-gated form is future
work that gets its own plan — nothing here supersedes the README as the
source of truth for what SimpleBroker does.
